# Stock Terminal

DIY Bloomberg-style desktop terminal for stock and ETF analysis.

## Dev

```bash
npm install
npm run dev
```

## Tech Stack

- **Electron** + **React** + **TypeScript** (via Vite)
- **FlexLayout** for docking panel layout
- **TradingView Lightweight Charts** for candlestick charts
- **Zustand** for state management
- **Tailwind CSS** for styling
- **cmdk** for command bar
